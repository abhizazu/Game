# 🚻 Urinal Etiquette

> *Can you navigate the unwritten rules of the men's restroom?*

A fun, fast-paced **social logic puzzle game** built with pure HTML, CSS & JavaScript — no libraries, no frameworks, just a single `index.html` file.

---

## 🎮 How to Play

1. You're shown a row of urinals — some are **occupied**, some are **free**
2. Pick the **best** available urinal based on real-world social etiquette
3. Score points for making the socially correct choice
4. Survive all **8 scenarios** and see your final rating!

### 🧠 The Social Rule
> Always maximize distance from other people. Never stand next to someone when a better spot is available.

---

## ✨ Features

- 🎯 **8 unique scenarios** with randomized layouts
- ⏱️ **Timer** — make your decision before time runs out
- 🎵 **Procedural audio** — sound effects for correct/wrong answers
- 🎉 **Confetti** on correct answers
- 📱 **Fully responsive** — works on phone, tablet, iPad, laptop & desktop
- 🖱️ **Hover glow** — highlights clickable urinals on desktop
- 🏆 **Rating system** — LEGEND / MAN / BOY / CONFUSED
- 🔊 **Drumroll** reveal for your final score

## 🚀 Run Locally

No build step needed — just open the file:

```bash

cd Game
open index.html
```

Or simply double-click `index.html` in Finder.
